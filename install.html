Redirecionando para o app em 15 segundos...

Instalação no Android

  1. Abra este link no Chrome: domineoviolaoapp.netlify.app
  2. Toque no banner "Adicionar à tela inicial" quando aparecer
  3. OU toque no menu () e selecione "Instalar aplicativo"
  4. Confirme a instalação tocando em "Instalar"
Abrir no Chrome

Instalação no iPhone/iPad

  1. Abra este link no Safari: domineoviolaoapp.netlify.app
  2. Toque no botão compartilhar ()
  3. Role para baixo e toque em "Adicionar à Tela de Início"
  4. Digite um nome (ou mantenha o sugerido) e toque em "Adicionar"
Abrir no Safari

Instalação no Computador

  1. Abra este link no Chrome: domineoviolaoapp.netlify.app
  2. Clique no ícone de instalação () na barra de endereços
  3. OU clique no menu () e selecione "Instalar Domine o Violão"
  4. Clique em "Instalar" na janela de confirmação
Abrir no Chrome

Após instalar, você verá o ícone do Domine o Violão na sua tela inicial para acesso rápido.
