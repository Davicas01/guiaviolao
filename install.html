Como instalar o aplicativo

No Android

  1. Abra o Chrome e acesse este site
  2. Toque no banner de instalação quando aparecer
  3. OU toque no menu () e selecione "Instalar aplicativo"
  4. Confirme tocando em "Instalar"

No iPhone/iPad

  1. Abra o Safari e acesse este site
  2. Toque no botão compartilhar ()
  3. Role para baixo e toque em "Adicionar à Tela de Início"
  4. Toque em "Adicionar" para confirmar

No Computador

  1. Abra o Chrome e acesse este site
  2. Clique no ícone de instalação () na barra de endereços
  3. OU clique no menu () e selecione "Instalar Domine o Violão"
  4. Clique em "Instalar" para confirmar

Após instalar, você terá acesso ao aplicativo diretamente da sua tela inicial!
